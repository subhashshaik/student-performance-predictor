# 🎓 Student Performance Predictor

This project predicts student exam scores using machine learning based on study habits and past performance.

## 🚀 Features
- Predicts final score using:
  - Study hours
  - Sleep hours
  - Attendance
  - Previous scores

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 How it works
A Linear Regression model is trained on student data to predict final exam scores.

## ▶️ How to Run
1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python model/model.py
